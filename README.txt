File URL Downloader:
1. Take URL as an input and download the target file.
2. File stored locally in device.
3. Local database keep track of the downloaded files.
4. Can view the files in jpg, png, mp3, wav, and mp4.
5. PDF cannot be viewed due to Unity not support it by default.
6. PDF can be viewed by using plugin for sale in Unity Asset Store.

Guide to build on Android
1. This project use Unity2021.2.7f1.
1. Ensure Unity build platform set to Android.
2. Connect device to PC.
3. Device should be in developer mode and allow PC to use debug mode.
4. Click Build and Run.
